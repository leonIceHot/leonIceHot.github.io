GPU Monitor

發表於 2026-01-22 更新於 2026-01-25 分類於 Computer Science ， Operating System ， Linux 文章字數： 478 所需閱讀時間 ≈ 1 分鐘

在 Ubuntu 環境中，監控 GPU 的狀態（使用率、記憶體 VRAM、溫度）非常關鍵，因為這能幫你判斷程式是否真的有吃到 GPU 加速，或者是否即將記憶體溢出（Out Of Memory）。

nvidia-smi

nvidia-smi (NVIDIA System Management Interface) 是一個基於 NVIDIA Management Library (NVML) 的命令列工具，專門用來管理和監控 NVIDIA GPU 裝置。

nvitop

nvitop 是一個專為終端機（Terminal）設計的互動式 NVIDIA GPU 行程檢視器。

nvtop

是基於 C 語言 開發的，速度極快，風格非常像 Linux 的 htop 指令，且支援 AMD 和 Intel GPU。

top

top 是 Linux 系統中最經典、最古老，也是幾乎所有發行版（Distro）都預設安裝的系統效能監控工具，監控 CPU (處理器) 和 RAM (記憶體)。

htop

htop 可以說是 top 的「現代化升級版」。它也是監控 Linux 系統 CPU 和記憶體最推薦的工具。

gpustat

gpustat 是另一個非常經典且極度簡潔的 NVIDIA GPU 狀態監控工具。
