VS Code 快速鍵指南

發表於 2022-04-22 分類於 IDE 文章字數： 645 所需閱讀時間 ≈ 1 分鐘

前言

為了迅速在VS Code 開發軟體，將一些常用的功能做一個整理，做一個備忘錄。
學著不使用滑鼠迅速開發。

面板設定

  • Ctrl + => 放大
  • Ctrl - => 縮小
  • Ctrl Tab => 分頁切換

編輯操作

  • Ctrl P : 10 => 切換至第10列
  • Ctrl Home => 至頁首
  • Ctrl End => 至頁尾
  • Ctrl L => 整段選取
  • Ctrl up down=> 位移移動
  • Ctrl / => 快速/解除註解
  • Ctrl + Shift + P =>

檔案操作

  • Ctrl A => 全選
  • Ctrl S => 存檔
  • Ctrl P => 選取檔案
