ORB-SLAM3 安裝指南

發表於 2022-05-15 更新於 2022-07-04 分類於 Computer Vision 閱讀次數： 文章字數： 2.4k 所需閱讀時間 ≈ 4 分鐘

前言

ORB-SLAM3 添加了視覺慣性處理，除了可用於單眼相機、立體相機、深度相機，亦支援針孔相機模型與魚眼相機模型等。
紀錄從beta版本更新至1.0的過程。

開發環境

Ubuntu 22.04 LTS 測試成功 (2022/05/06)
Ubuntu 20.04 LTS
ORB-SLAM3 v1.0 - https://github.com/UZ-SLAMLab/ORB_SLAM3

預先安裝

sudo apt-get install git
sudo apt-get install vim
sudo apt-get install g++
sudo apt-get install gcc
sudo apt-get install cmake
sudo apt-get install libssl-dev # openssl
sudo apt-get install zip
sudo apt-get install sudo

安裝Eigen3

sudo apt-get install libeigen3-dev

安裝Pangolin

必要選項
sudo apt install libgl1-mesa-dev (opengl)
sudo apt-get install libglew-dev (glew)
sudo apt-get install libboost-dev libboost-thread-dev libboost-filesystem-dev
sudo apt-get install libpython2.7-dev libpython3-dev
sudo apt-get install build-essential

從github下載v0.6
git clone https://github.com/stevenlovegrove/Pangolin/tree/v0.6

編譯安裝
cd Pangolin
mkdir build
cd build
cmake ..
make -j 20 # 數字代表執行緒
sudo make install

安裝OpenCV

新版 - OpenCV 4.4
https://github.com/opencv/opencv/archive/4.4.0.zip

修正OpenCV ver.4.4 Error
opencv/modules/gapi/test/gapi_async_test.cpp 添加兩行

舊版 - OpenCV 3.4
https://github.com/opencv/opencv/archive/3.4.zip

必要選項
[compiler]
sudo apt-get install build-essential
[required]
sudo apt-get install libgtk2.0-dev libgtk-3-dev libavcodec-dev libavformat-dev libswscale-dev
[optional]
sudo apt install python-dev python-numpy python3-dev python3-numpy

編譯安裝

安裝g2o

[required]
sudo apt-get install libeigen3-dev libsuitesparse-dev qtdeclarative5-dev qt5-qmake libqglviewer-dev

cd Thirdparty/g2o
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j 8

安裝DBoW2

[required]
sudo apt-get install libboost-dev

cd Thirdparty/DBoW2
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j 8 # runs 8 jobs in parallel

安裝 Sophus

cd Thirdparty/Sophus
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j 8 # runs 8 jobs in parallel

ORB-SLAM3 編譯

cd ORB_SLAM3
chmod +x build.sh
./build.sh
