ORB-SLAM3 安裝指南

發表於 2022-05-15 更新於 2024-12-11 分類於 Computer Science ， Computer Vision 文章字數： 5.7k 所需閱讀時間 ≈ 10 分鐘

前言

ORB-SLAM3 1. IMU (Inertial measurement unit) 慣性測量單元 2. 相機: 1. 單眼相機 2. 立體相機 3. 深度相機 4. 針孔相機 5. 魚眼相機 3. 更新過程 beta 0.4 ~ 1.0 realease。

背景介紹

视觉SLAM原理与ORB-SLAM3系列算法-视觉SLAM原理与ORB-SLAM3系列算法 https://www.youtube.com/watch?app=desktop&v=LZ-gKdUDwos&ab_channel=%EB%B0%94%EC%B9%B4%EB%9D%BC%ED%99%A9%EC%A0%9C14

開發環境

  • ROS2 Humble
  • Ubuntu 22.10 Kinetic Kudu
  • ORB-SLAM3 v1.0 https://github.com/UZ-SLAMLab/ORB_SLAM3
  • ORB-SLAM3 detailed comments https://github.com/electech6/ORB_SLAM3_detailed_comments
  • ORB_SLAM3_ROS2 https://github.com/zang09/ORB_SLAM3_ROS2

預先安裝

sudo apt-get install git sudo apt-get install vim sudo apt-get install g++ sudo apt-get install gcc sudo apt-get install cmake sudo apt-get install libssl-dev # openssl sudo apt-get install zip sudo apt-get install sudo

安裝 Eigen3

sudo apt-get install libeigen3-dev

安裝 Pangolin

必要選項

sudo apt install libgl1-mesa-dev (opengl) sudo apt-get install libglew-dev (glew) sudo apt-get install libboost-dev libboost-thread-dev libboost-filesystem-dev sudo apt-get install libpython2.7-dev libpython3-dev sudo apt-get install build-essential

從github下載v0.6

git clone https://github.com/stevenlovegrove/Pangolin/tree/v0.6

編譯安裝

cd Pangolin mkdir build cd build cmake .. make -j 20 # 數字代表執行緒 sudo make install

Pangolin Compile Error

error : ‘numeric_limits’ is not a member of ‘std’

安裝 g2o

[required] sudo apt-get install libeigen3-dev libsuitesparse-dev qtdeclarative5-dev qt5-qmake

cd Thirdparty/g2o mkdir build cd build cmake .. -DCMAKE_BUILD_TYPE=Release make -j 8

安裝 DBoW2

[required] sudo apt-get install libboost-dev

cd Thirdparty/DBoW2 mkdir build cd build cmake .. -DCMAKE_BUILD_TYPE=Release make -j 8 # runs 8 jobs in parallel

安裝 Sophus

cd Thirdparty/Sophus mkdir build cd build cmake .. -DCMAKE_BUILD_TYPE=Release make -j 8 # runs 8 jobs in parallel sudo make install
