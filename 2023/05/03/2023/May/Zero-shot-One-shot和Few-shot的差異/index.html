Zero-shot, One-shot和Few-shot的差異

發表於 2023-05-03 更新於 2024-12-11 分類於 Computer Science ， Artificial Intelligence ， Machine Learning ， Transfer Learning 文章字數： 2.5k 所需閱讀時間 ≈ 5 分鐘

簡介

Zero-shot learning（零樣本學習）、One-shot learning（單樣本學習）和Few-shot learning（少樣本學習）是三種在限制樣本情境下的機器學習方法，它們之間有以下的差異：

  1. 樣本數量：Zero-shot learning 是在完全沒有樣本的情況下進行學習，模型需要從類別的描述或屬性中推理和學習。One-shot learning 是在僅有一個樣本的情況下進行學習，模型需要利用這一個樣本進行推理和識別。Few-shot learning 是在僅有少量樣本的情況下進行學習，通常是數個或數十個樣本。

  2. 物體任務：Zero-shot learning 的目標是識別或推理之前從未見過的類別，模型需要從類別的描述或屬性中學習如何識別這些類別。One-shot learning 的目標是通過僅有的一個樣本進行類別識別，模型需要學習如何將這個樣本與新的樣本進行比較。Few-shot learning 的目標是通過少量樣本學習如何適應新的任務或類別，模型需要在有限的樣本上進行訓練和泛化。

  3. 方法和策略：Zero-shot learning 使用屬性推理、轉移學習或語義嵌入等方法來利用類別的描述或屬性進行學習和識別。One-shot learning 通常使用相似度比較、生成模型或概念學習等方法來從僅有的一個樣本進行學習和識別。Few-shot learning 則利用元學習、生成模型或轉移學習等方法來在少量樣本上進行訓練和泛化。

總體而言，Zero-shot learning 是在沒有任何樣本的情況下學習和識別新的類別，One-shot learning 是在僅有一個樣本的情況下學習和識別，Few-shot learning 是在少量樣本的情況下學習和識別。

Zero-shot learning

Zero-shot learning（零樣本學習）是指在沒有任何訓練樣本的情況下，訓練機器學習模型來識別新的類別或任務。傳統的機器學習方法需要使用大量的樣本數據進行訓練，但在現實應用中，可能會遇到之前未見過的類別或任務，而這些類別在訓練集中是不存在的。

Zero-shot learning 的目標是使模型能夠通過理解和應用類別之間的關係和屬性，對新的類別進行準確的識別。為了實現這一目標，常見的 zero-shot learning 方法包括：

  1. 屬性推理（Attribute Inference）：這種方法基於對類別屬性的推理。模型學習到類別的屬性，例如顏色、形狀、大小等，然後通過屬性推理來判斷新的類別。

  2. 轉移學習（Transfer Learning）：轉移學習是指將從其他相關任務或類別中學到的知識和特徵轉移到新的任務或類別中。在 zero-shot learning 中，轉移學習可以利用已經訓練好的模型的知識和特徵，來進行新類別的識別。

  3. 語義嵌入（Semantic Embedding）：語義嵌入方法將類別映射到一個低維度的向量空間，使得類別之間的關係可以用向量之間的距離或相似度來表示。模型通過學習類別的語義嵌入，可以將新的類別映射到該空間中，並根據其在空間中的位置進行識別。

One-shot learning

One-shot learning（單樣本學習）是指在只有一個樣本的情況下，訓練機器學習模型來識別新的類別或任務。在傳統的機器學習中，通常需要大量的樣本數據才能訓練出準確的模型，然而在某些情況下，我們可能只有一個樣本可用，例如罕見的類別、特殊的情境或稀有的事件。

One-shot learning 的目標是使模型能夠從僅有的一個樣本中學習到足夠的知識和特徵，以便在面對新的類別或任務時進行準確的識別或推理。為了實現這一目標，常見的 one-shot learning 方法包括：

  1. 相似度比較（Similarity Comparison）：這種方法通常基於樣本之間的相似度計算進行識別。模型學習到如何比較新樣本與已知類別樣本之間的相似度，並根據最相似的樣本進行分類。

  2. 概念學習（Concept Learning）：概念學習方法旨在學習到類別的通用特徵或屬性。模型通過一個概念的例子來學習該類別的特徵，並根據這些特徵來識別新的類別。

  3. 生成模型（Generative Models）：生成模型是指通過學習數據的分布來生成新的樣本。在 one-shot learning 中，生成模型可以用來生成額外的樣本數據，以增加訓練數據的多樣性，從而提高模型的識別能力。

  4. 元學習（Meta-Learning）：元學習方法可以學習到如何快速適應新的任務。在 one-shot learning 中，元學習模型通過先前的學習經驗和知識，快速適應新的類別或任務。

Few-shot learning

Few-shot learning（少樣本學習）是指在僅有少量樣本（通常是數個或數十個）的情況下，訓練機器學習模型來處理新的任務或類別。在傳統的機器學習中，通常需要大量的樣本數據才能訓練出準確的模型，然而在現實應用中，我們可能只有很少的樣本可用，而這使得傳統的學習方法難以達到良好的性能。

Few-shot learning 的目標是使模型能夠從有限的樣本中學習到泛化能力，並能夠在面對新的類別或任務時表現良好。為了實現這一目標，常見的 few-shot learning 方法包括：

  1. 元學習（Meta-Learning）：元學習是指訓練一個模型來學習如何快速適應新任務。在 few-shot learning 中，元學習模型會通過從多個相似的少樣本學習任務中學習到共享的特徵和知識，然後使用這些學習到的知識來快速適應新的類別或任務。

  2. 生成模型（Generative Models）：生成模型是指通過學習數據的分布來生成新的樣本。在 few-shot learning 中，生成模型可以用來生成額外的樣本數據，以增加訓練數據的多樣性，從而提高模型的泛化能力。

  3. 轉移學習（Transfer Learning）：轉移學習是指將從其他相關任務或類別中學到的知識和特徵轉移到新的任務或類別中。在 few-shot learning 中，轉移學習可以利用已經訓練好的模型的知識和特徵，快速適應新的類別或任務。
