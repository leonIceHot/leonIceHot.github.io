React 基本功能備忘錄

簡介

React 是一個由 Facebook 開發的 JavaScript 函式庫，用於建立使用者界面。它主要用於開發單頁應用程序（Single Page Application）和行動應用程式。
React 通過使用元件（Component）來建立使用者界面，並且可以通過虛擬 DOM（Virtual DOM）技術來實現高效的 UI 更新和渲染。

React 的主要特點包括：

  1. 元件化：React 將 UI 拆分為多個元件，每個組件都可以獨立設計和開發，使代碼更加模組化和可重用。
    虛擬 DOM：React 使用虛擬 DOM 技術來實現高效的 UI 更新和渲染。當狀態（State）發生變化時，React 會計算出虛擬 DOM 中需要更新的部分，然後只更新這些部分，從而提高性能。
  2. 單向資料流：React 推崇單向資料流的設計思想，即數據只能從父級元件流向子級組件，而不能反向流動，從而簡化了代碼的設計和調試。
  3. JSX 語法：React 使用 JSX 語法來描述組件的 UI，這種語法類似於 HTML，但可以在其中嵌入 JavaScript 代碼，從而實現更加靈活和強大的 UI 設計。
  4. React 已成為當前最受歡迎的前端框架之一，擁有廣泛的開發者社區和生態系統，可以應用於各種 Web 和移動端開發專案
