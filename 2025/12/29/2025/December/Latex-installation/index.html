Local installation

Compiler Error : LaTeX Error: File `stfloats.sty’ not found.

Solution

Place the .sty file in your project folder
方法 A（建議）：從 CTAN 下載已打包好的 .sty
到 CTAN： https://ctan.org/pkg/sttools
點 Download（zip）後解壓縮
在解壓後的內容裡應該能找到 stfloats.sty
把 stfloats.sty 放到你的主 .tex 同一個資料夾，再重新編譯
如果你下載的那包只有 .dtx，通常是你拿到「來源檔」而不是「已生成檔」。

方法 B：用 .dtx 自己生成 stfloats.sty
在 Windows（MiKTeX）照做：

建一個資料夾，例如 C:\temp\stfloats
把 stfloats.dtx 複製到這個資料夾
在該資料夾空白處 Shift + 右鍵 → 開啟 PowerShell/終端機
依序執行：
先解出檔案（會生成 stfloats.ins）：

再用 .ins 生成 .sty：

tex stfloats.ins
完成後，資料夾裡應該會出現 stfloats.sty。

把生成的 stfloats.sty 複製到你的專案主檔 .tex 同一層，重新編譯。
