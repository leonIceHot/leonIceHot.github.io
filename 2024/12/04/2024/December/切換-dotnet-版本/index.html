安裝完成後，您可以使用 dotnet --list-sdks 來檢查是否成功安裝。

6. 在 Visual Studio 中切換 .NET 版本（如果適用）

  1. 右鍵點擊專案 > 屬性。
  2. 在 Application 或 應用程式 分頁中，修改 Target Framework。
  3. 選擇已安裝的 .NET 版本（例如 .NET 8.0）。
