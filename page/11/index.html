Docsify 是一個基於 JavaScript 的文件網站生成器，它可以幫助您快速創建美觀、易於瀏覽的文件網站。Docsify 可以將 Markdown 文件轉換為一個動態的單頁應用程式，這意味著您可以通過單頁應用程序的方式來瀏覽您的文件，而不需要每次點擊頁面時都重新加載頁面。

Docsify 非常易於使用，只需將 Markdown 文件放在您的專案中，然後在您的 HTML 文件中引入 Docsify 腳本。Docsify 會自動將您的 Markdown 文件轉換為一個動態的單頁應用程序。您可以使用 Docsify 的設置選項來自定義您的文件網站的外觀和行為。Docsify 還提供了一些插件，如搜索、目錄等，以增強您的文件網站的功能。

Docsify 的主要優點是它的輕量級和簡單易用。它不需要任何複雜的配置或安裝過程，您只需將文件拖放到您的專案中並引入 Docsify 腳本即可。Docsify 的文件網站看起來非常專業和現代，因此它是一個非常受歡迎的選擇，特別是對於那些想要快速搭建文件網站的開發人員來說。

簡介

OAuth（Open Authorization）是一個開放標準的授權協議，用於授權第三方應用程序訪問用戶資源，並保護用戶的資源免於被盜用。OAuth 協議可以讓用戶授權第三方應用程式訪問其資源，而不需要將用戶名稱和密碼直接提供給第三方應用程式，從而保護用戶的隱私和安全性。

OAuth 2.0 是一個用於授權的開放標準，旨在簡化基於網路的應用程式對第三方服務的訪問控制。相比於 OAuth 1.0，OAuth 2.0 更為簡潔，易於實現和使用，並且支持多種授權類型。

SASS/SCSS 基本語法實作

發表於 2023-04-18 更新於 2024-12-11 分類於 Software Engineering ， Web Development 文章字數： 492 所需閱讀時間 ≈ 1 分鐘

SASS（Syntactically Awesome Style Sheets）和 SCSS（Sassy CSS）都是 CSS 的擴展語言，可以在 CSS 的基礎上增加一些功能，使得 CSS 更加易讀、易寫、易維護。

SASS 是一種以 Ruby 為基礎的語言，它使用了類似 Ruby 的語法和許多特性，例如變量、嵌套規則、Mixin、函數等，可以大大簡化 CSS 的編寫。使用 SASS，可以定義變量，方便地管理顏色、字體大小等常用屬性，同時可以使用 Mixin 和函數，實現代碼的重用和結構的組織。不過，SASS 的語法需要學習，並且需要安裝 Ruby 環境。

SCSS 則是 SASS 的一種新的語法，它基於 CSS3 的語法，使用了大括號、分號等符號，更接近 CSS 的語法，因此可以更容易地學習和使用。同樣支持變量、嵌套規則、Mixin、函數等功能，並且可以使用現有的 CSS 代碼。使用 SCSS，可以在不修改現有 CSS 的情況下進行擴展和改進。

總的來說，SASS 和 SCSS 都是非常有用的工具，可以讓 CSS 更加易讀、易寫、易維護，並且提高開發效率和代碼重用性。如果您還沒有使用過它們，建議可以了解一下它們的基本語法和用法，並在實際開發中進行應用。
